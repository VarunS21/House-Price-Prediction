# Home Pick

Home Pick is a cutting-edge machine learning model designed to predict house prices in Chennai with remarkable accuracy. It harnesses historical data, current market trends, and a variety of features such as location, property size, and amenities to provide users with reliable price estimates. This model is particularly tailored for Chennaiâ€™s unique real estate landscape, making it an invaluable tool for buyers, sellers, and real estate professionals looking to make informed decisions in the vibrant Chennai housing market. ### INSTALLATION

```
git clone https://github.com/VarunS21/House-Price-Prediction.git
```

```
cd home-pick
```

```
python -m pip install -r requirements.txt

```

```
.venv\Scripts\activate
```

### Execution

Run the application by double clicking the 'run.bat' file

or

```
python app.py
```
